# geoRisk
This code deals with disclosure risk for geo-spatial data in relation to thematic maps.

It gives C++ code to calculate the minimal _k_-anonymity regions for the point-oriented and the cluster-oriented risk measure.

There is also an R script that was used to produce the results mentioned in the paper _Disclosure risk in a geo-spatial setting_ (2026), submitted to PSD2026 and prepublished on [arxiv.org](https://arxiv.org/abs/2606.13327)
